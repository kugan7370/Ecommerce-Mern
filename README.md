

# Ecommerce APP






## Features

    Login
    Register
    Cart Options
    Wishlist Options
    File upload using Multer
    

    
    


## Environment Variables

To run this project, you will need to add the following environment variables to your .env file

`Mongo_Connect`

`JWT_SECRET`

`CLOUD_NAME`

`CLOUD_API_Key`

`CLOUD_API_SECRET`


## Tech Stack

**Client:** React, Redux toolkit

**Server:** Node, Express

**DB**: Mongo Atlas, Mongoose
